# 孽主推线与抗压思路

孽主是一个特殊的**推线型三号位**。他的核心打法不是单杀，而是通过强大的推线能力，让对面始终处于防守压力之下。

## 对线期：稳如老狗

孽主的对线能力不算顶尖，但胜在**稳定**。火雨（Firestorm）可以远程清兵，被动（Atrophy Aura）让对面核心补刀很难受。

**核心思路：** 推线 → 拉野/囤野 → 推线 → 循环

不要让兵线一直压在自己塔下。用火焰风暴把线推过去，然后去拉野或者支援其他路。

## 推线节奏

孽主的火焰风暴（Firestorm）是DOTA2里最强的推线技能之一。**你的目标不是单杀对面大哥，而是让对面大哥不敢离开自己的塔。**

**正确的推线节奏：**
1. 火焰风暴清兵 → 兵线推到对面塔下
2. 对面核心被迫守塔，无法去别的地方刷钱
3. 你利用这段时间去拉野、做眼、或者支援中路
4. 兵线回来了，重复以上步骤

## 大招（Fiend's Gate）的战略价值

孽主的大招是全图传送门，这是DOTA2里最强的大局技能之一。

**使用场景：**
- 队友在另一条路打起来了，开大招带人支援
- 偷盾：开大招把全队传到肉山坑
- 推高地：开大招让全队从奇怪的角度切入
- 逃跑：被追杀时，在树林深处开大，传送到安全位置

**关键：** 大招的落点要选在有视野、安全的地方。不要开在对面脸上。

## 装备选择

孽主的核心装备是**团队装 + 肉装**：
1. **秘法鞋/相位鞋**：都可以，缺蓝出秘法
2. **赤红甲**：配合火焰风暴，让对面物理核心更难受
3. **笛子/莲花**：根据对面阵容
4. **冰甲/龙心**：后期补肉

**误区：** 不要出输出装。孽主的输出来自于被动偷的攻击力和火焰风暴，不需要额外的输出装备。

## 常见错误

- **火焰风暴只用来打人，不清兵**：火焰风暴的主要作用是推线，打人只是附带效果
- **大招开得随意**：大招CD很长，每一次开启都应该有明确的目的
- **过于执着于单杀**：孽主不是单杀英雄，推线和团队节奏更重要

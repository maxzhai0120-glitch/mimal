# 末日使者单抓与大招选择

末日使者是一个**单抓型+压制型**的三号位。他的核心技能——末日（Doom），是DOTA2里最强力的单体沉默技能之一。

## 对线期：压制与发育

末日的对线能力中等，但因为有吞噬（Devour）可以吃野怪获得技能，所以对线期有一定的变数。

**对线技巧：**
- 吞噬有光环的野怪（如狼的加攻击光环、枭兽的减速光环），增强对线能力
- 焦土（Scorched Earth）可以用来清兵+回血+加速，但不要无脑用来推线
- 6级后有了末日，配合辅助可以稳定击杀对面核心

## 末日（Doom）的目标选择

**末日是DOTA2里最强力的单体控制技能**，但CD很长（100秒以上），所以每一次释放都要慎重选择目标。

**优先级：**
1. **对面关键辅助**：神谕、戴泽、暗影恶魔等。这些英雄的技能可以救队友，给他们套末日可以确保击杀
2. **对面开团英雄**：潮汐、猛犸、谜团等。给他们套末日，让他们放不出大招
3. **对面高输出核心**：如果对面核心没有BKB，给他套末日可以直接废掉他8-16秒
4. **对面逃生英雄**：敌法师、女王、蚂蚁等。给他们套末日，让他们无法逃生

**注意：** 末日可以被林肯抵挡，可以被莲花反弹。释放前注意观察。

## 单抓节奏

末日6级后，是**单抓能力最强的三号位之一**。

**单抓流程：**
1. 开启焦土（加速+回血）
2. 追上目标
3. 给末日（沉默+持续伤害）
4. 追着A，等目标死亡

**单抓目标优先级：**
- 对面打野的核心（敌法师、幽鬼等）
- 对面单独带线的英雄
- 对面做眼的辅助

## 装备选择

1. **相位鞋/秘法鞋**：都可以，需要追人出相位
2. **点金手**：末日刷钱很快，点金可以进一步加速发育
3. **跳刀**：增加先手能力，但不是必出
4. **BKB**：末日给对面后，自己也要防止被控
5. **A杖**：质变，末日持续时间更长，且破坏被动技能

## 常见错误

- **末日给了无关紧要的人**：比如给了一个5号位辅助，结果对面大哥把你队友杀光了
- **单抓时过于深入**：追人追到对面二塔下，结果被反杀
- **忽视了吞噬技能的选择**：不同的野怪技能适合不同的局势，不要无脑吃第一个看到的野怪
